Introduction

  • This app uses the Credit data set from ISLR package.
  • It analyzes the relationships between Income and other predictors.
  • It presents an input button for “student” variable indicator, and two sliders for “balance” and “rating” variable inputs for prediction.
  • It plots the residuals of the regression model of formula = Income ~ Rating*Balance + Student

Exploratory Analysis

The model is selected from the analysis of predictors in the Credit dataset.

pairs(Credit)

Selecting Predictors

The model begins with the most significant predictors, ‘StudentYes’, ‘Balance’ and ‘Limit’. Upon further examination, because ‘Rating’ and ‘Limit’ is very highly correlated with a value of 0.9968797 , ‘Rating’ predictor is used instead of ‘Limit’ because it makes more sense.
